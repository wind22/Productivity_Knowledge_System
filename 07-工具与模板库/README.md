# 07 工具与模板库

> 本章是知识库的「工具箱」：前面章节讲原理与方法，这里给可以直接抄走用的东西。所有模板均**拿来即用**——结构完整、每个字段带填写说明（`<!-- -->` 注释或斜体示例占位）、附填写示例。
>
> 使用方式：复制对应文件到你的工作目录/协作平台，按注释填写，删掉注释与示例即成正式文档。表格类模板可原样搬进飞书多维表格或 Excel。

---

## 一、文档模板

写方案、写报告时直接套用的完整文档骨架。

| 模板 | 一句话用途 |
|---|---|
| [PRD模板](模板/PRD模板.md) | 产品需求文档：从背景目标到灰度回滚的全结构，含埋点事件表与 Given-When-Then 验收标准 |
| [竞品分析模板](模板/竞品分析模板.md) | 以决策问题为锚的竞品分析：定位/功能矩阵/数据对比 + 策略推演，强制输出 So What |
| [活动方案模板](模板/活动方案模板.md) | 运营活动完整方案：量化目标、玩法规则、成本预算与熔断、风控预案、责任分工 |
| [复盘报告模板](模板/复盘报告模板.md) | 五步复盘：目标回顾→结果对比→过程还原→归因分析→经验沉淀为可复用 SOP |
| [商业计划书模板](模板/商业计划书模板.md) | BP 结构：问题→方案→市场（TAM/SAM/SOM）→竞争→商业模式→团队→财务预测→融资计划 |
| [分析报告模板](模板/分析报告模板.md) | 结论先行的数据分析报告：TL;DR→关键发现→数据支撑→行动建议，附口径说明附录 |

## 二、表格模板

日常持续维护的台账与工作表，均附字段说明与示例行。

| 模板 | 一句话用途 |
|---|---|
| [指标字典模板](表格/指标字典模板.md) | 团队指标口径的唯一事实来源：定义/计算口径/SQL 示例/负责人，附 DAU、次留等 5 个示例 |
| [需求池模板](表格/需求池模板.md) | 需求统一入口与 RICE 优先级打分表，附打分速查与反模式提醒 |
| [渠道ROI台账模板](表格/渠道ROI台账模板.md) | 投放渠道周度台账：花费/激活/次留/CAC/LTV/ROI，附计算公式与放缩量决策建议 |
| [单位经济模型测算表](表格/单位经济模型测算表.md) | 单均收入-成本-毛利结构 + LTV/CAC 测算与健康度判断（>3、回收周期），附外卖平台完整示例 |
| [AB测试记录表](表格/AB测试记录表.md) | 实验登记表：假设/样本量/护栏指标/结果/决策/经验沉淀，附实验常见坑自查 |

## 三、检查清单

上线前、核对时逐项打勾的 checkbox 清单，防"想当然"。

| 清单 | 一句话用途 |
|---|---|
| [功能上线前检查清单](清单/功能上线前检查清单.md) | 功能上线七查：需求验收、埋点、兼容性能、灰度开关、回滚、监控、协作通知 |
| [活动上线前检查清单](清单/活动上线前检查清单.md) | 活动上线七查：目标指标、规则文案、成本审批、风控防刷、压测、客服口径、应急预案 |
| [数据口径核对清单](清单/数据口径核对清单.md) | 数字对不上时的排查清单：定义、周期、去重、过滤、分母、时区、埋点时机、历史可比性 |
| [商业化风控清单](清单/商业化风控清单.md) | 涉钱功能的攻击者视角推演：激励漏洞、薅羊毛路径、预算熔断、价格与概率合规、客诉预案 |

| 做实验、出分析结论 | [AB测试记录表](表格/AB测试记录表.md) + [分析报告模板](模板/分析报告模板.md) |
| 写融资材料 | [商业计划书模板](模板/商业计划书模板.md) |
