## 4. 归因分析

<!-- 对每个「结果与目标的差距」做归因。要求：
1. 区分「可控因素」与「不可控因素」——只有可控因素能产生行动；
2. 用「五个为什么」追到过程/机制层面，不停留在"执行不到位"；
3. 归因要有数据证据，不接受"我感觉"；
4. 做得好的也要归因——好结果如果只是运气，下次复制不了 -->

### 4.1 留存未达标（33.2% vs 35%）

- **数据证据**：*漏斗显示连签 3 天后流失陡增；分层看，仅为红包而来的低质渠道用户（渠道 X）7 日留存仅 19%，拉低大盘约 2.1pp。*
- **五个为什么**：*为什么留存低 → 渠道 X 用户签完即走 → 为什么引入渠道 X → 爆发期追求参与量临时加投 → 为什么临时加投没评估质量 → 投放决策只看 CPA 不看留存预估 → **根因：投放放量决策缺少"分渠道留存预估"这一必经环节（机制缺失，可控）**。*

### 4.2 成本超支 9.2%

- **数据证据**：*超支集中在 08-25~08-27，参与量为预估 1.6 倍。*
- **归因**：*参与率预估基于去年同期，未纳入今年竞品同期缺席的因素（不可控 40%）；预算熔断线设为日预算 150% 过于宽松，触发时已超支（可控 60%）。*

### 4.3 做得好的：风控预案生效

- *团伙账号被拦截，避免约 8 万元资损。原因：上线前完成了薅羊毛路径推演并预置了设备指纹规则——此流程值得固化为标准动作。*
