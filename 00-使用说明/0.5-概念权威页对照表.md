# 0.5 概念权威页对照表

> 📍 所属：第 0 章 使用说明 · 解决"同一概念在多章出现，更新时改哪里"的口径漂移问题。

## 为什么需要权威页

本库覆盖四个领域，RFM、LTV/CAC、A/B 测试这类概念天然横跨多章。如果每处都独立展开，日后更新时极易出现"改了 3.2 忘了 4.3"的口径漂移。因此约定：

1. **每个跨领域概念只在一处深入定义与展开**——该处即"权威页"。
2. **其他章节只做一句话概述 + 链接到权威页**，不重复展开细节。
3. **更新概念时只改权威页**；引用处除非概述本身错了，否则不动。
4. 存量文档中已有的重复展开**不要求立即回收**，在日常修订触及时逐步收敛（渐进式，与 [0.2 知识管理方式](0.2-知识管理方式.md) 的渐进式总结一致）。
5. 新增跨章概念时，**先在本表登记权威页**，再动笔。

## 维护规则

- 本表是**唯一登记处**：新增/迁移权威页必须更新本表，并在提交信息中注明（前缀 `polish:`）。
- 拿不准某概念归属时的判定顺序：**它属于哪一层？**（心理学/思维类归 01；做法归各领域方法论；跨域组合归 06）→ **谁的读者最需要完整版？**
- 每季度随 [0.3 更新机制](0.3-更新机制.md) 的淘汰复查一起，抽查 3-5 个概念的引用处是否与权威页口径一致。
