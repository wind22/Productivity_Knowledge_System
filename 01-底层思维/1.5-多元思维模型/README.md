# 1.5 多元思维模型专题

> 📍 所属：第1章 底层思维层 · 本专题系统整理查理·芒格"多元思维模型格栅"（latticework of mental models）体系下、被多个权威来源共同认可的约 180 个思维模型，按学科分为 9 类，每个模型给出**定义 → 适用场景 → 带来的思考 → 局限性 → 关联模型**五要素。

## 专题定位

芒格的核心主张：**不能只用一个学科的工具看世界**——"手里只有锤子的人，看什么都像钉子"。他认为掌握约 80~90 个跨学科的重要模型，编成一张相互印证的"格栅"，就能覆盖成为"具备普世智慧的人"所需分量的 90%。本专题把这些模型系统落库，作为 [1.2 通用心智模型](../1.2-通用心智模型.md)（七个模型的互联网业务应用）的**全景扩展版**。

- **从哪读起**：先读 [1.5.1 总纲](1.5.1-总纲-格栅与使用法则.md) 理解"为什么要多模型并用"，再读 [1.5.2 通用思维概念](1.5.2-通用思维概念.md)（共识度最高的 12 个），其余 7 篇按需查阅。
- **时间有限**：只看总纲里"最高共识核心约 20 个模型"清单。
- **与 1.2 的分工**：重叠概念（系统/杠杆/边际/复利/概率/博弈/二阶）的业务应用以 1.2 为权威页，本专题给全景名单与模型关联。
