# 1.4 用户心理学基础

> 📍 所属：第1章 底层思维层 · 本文解决"用户为什么会点/买/留/走"的问题——所有产品设计、运营活动、转化优化背后共同依赖的心理学原理。

## 一、Fogg 行为模型：B = MAP

### 是什么

斯坦福大学 BJ Fogg 提出：**一个行为发生，当且仅当动机（Motivation）、能力（Ability）、提示（Prompt）同时具备**：

```
B = M × A × P
行为 = 动机 × 能力 × 提示
```

- **动机**：用户想不想做（追求快乐/逃避痛苦、追求希望/逃避恐惧、追求认同/逃避排斥）；
- **能力**：做这件事的难度——时间、金钱、体力、脑力、社交偏离度、是否违背习惯，六项中最短的板决定难度；
- **提示**：在正确时机出现的触发信号（Push、红点、按钮、场景联想）。

关键洞察：**动机与能力可以互相补偿**——动机极高时用户能忍受复杂流程（抢演唱会票），动机一般时流程必须极简。三要素缺一行为就不发生，且**提升能力（简化）通常比提升动机（说服）更便宜、更可靠**。

### 如何应用

转化率问题的标准诊断顺序：

1. 先查 **P（提示）**：用户在有动机的时刻收到入口了吗？提示被淹没了吗？
2. 再查 **A（能力）**：数一数完成行为需要几步、几次输入、几个决策点，砍到不能再砍；
3. 最后才做 **M（动机）**：补贴、文案、稀缺感——最贵且最易衰减的一档。

| 手段 | 作用的变量 | 示例 |
|------|-----------|------|
| 微信"一键登录"替代注册表单 | A↑ | 手机号一键授权，把 6 步注册砍成 1 步 |
| 拼多多"仅差 0.9 元可提现" | M↑ | 目标临近感放大动机 |
| 美团饭点前的 Push"今天吃什么" | P（时机） | 提示精确落在动机峰值时刻 |

### 示例：抖音的极简闭环

抖音把"消费一条内容"的行为做到 B=MAP 满分：打开即自动播放（P 与 A 合一，零决策）、上滑即下一条（A 极高）、内容由算法保证命中兴趣（M 稳定供给）。对比早期长视频 App：打开 → 选分类 → 挑封面 → 点播放 → 等广告，每一步都在漏损。**能力侧的差距，是抖音时长碾压的第一因**。

## 二、Hook 上瘾模型：习惯是怎么养成的

### 是什么

Nir Eyal 在《Hooked》中提出，让用户形成使用习惯需要反复经历四阶段循环：

```
触发（Trigger）→ 行动（Action）→ 可变奖励（Variable Reward）→ 投入（Investment）→（回到触发）
```

1. **触发**：外部触发（Push/红点）逐步过渡到内部触发（无聊→刷抖音，好奇→上知乎）；
2. **行动**：在触发后最简单的那个行为（B=MAP 的应用）；
3. **可变奖励**：奖励不可预测才持续有效（详见本文第六节）；
4. **投入**：让用户在产品里存入点什么——数据、关系、声誉、习惯——投入越多，下次触发越有效，替换成本越高（呼应 [1.1 用户价值公式](./1.1-商业第一性原理.md) 的替换成本项）。

### 如何应用

给自己的产品画 Hook 循环体检表：

- [ ] 用户的**内部触发**是什么情绪/场景？（没有内部触发的产品永远依赖买量和 Push）
- [ ] 从触发到核心行动是否 ≤ 3 步？
- [ ] 奖励是否可变？（固定奖励会边际效用递减，见 [1.2 边际思维](./1.2-通用心智模型.md)）
- [ ] 用户每次使用留下了什么**投入**？这些投入如何改善下一次体验？

### 示例：小红书的 Hook 循环

触发：想买东西怕踩雷（内部触发"决策焦虑"）→ 行动：搜索/刷信息流（一步到位）→ 可变奖励：每次刷到的笔记质量、风格不可预测，偶有"救命，这也太好用了"级别的惊喜 → 投入：收藏夹、关注博主、自己发笔记攒赞。用户收藏的每一篇攻略都是存进产品的资产，也是下一次"要买东西先搜小红书"的理由。

## 三、四大经典认知偏差：损失厌恶、锚定、沉没成本、禀赋效应

### 是什么与如何应用

这四个偏差同源于卡尼曼与特维斯基的前景理论（Prospect Theory），是转化与留存设计中使用频率最高的心理机制：

| 偏差 | 定义 | 强度/特征 | 典型应用（示例） |
|------|------|----------|----------------|
| **损失厌恶** | 损失带来的痛苦约为同等收益快乐的 2-2.5 倍 | 人对"失去"远比"得不到"敏感 | 淘宝"限时券 2 小时后过期"；会员到期前"你将失去 8 项特权"的挽留弹窗；连续签到中断清零 |
| **锚定效应** | 第一个出现的数字成为后续判断的参照点 | 即使锚明显不合理也起作用 | 划线价"¥299 ¥99"；直播间"原价三位数，今天只要 39.9"；SaaS 定价页把最贵档放最左边 |
| **沉没成本** | 已投入且不可收回的成本，非理性地影响后续决策 | 理性上应忽略，行为上很难 | 视频会员"再看 3 天就到期，续费不浪费"；游戏里已练到 58 级的账号让人无法弃坑；美团月卡买了就想"用回本" |
| **禀赋效应** | 一旦拥有某物，对它的估值立刻升高（约 2 倍） | "拥有"可以是虚拟/试用的 | 7 天免费试用会员（到期扣款）；网易云音乐年度报告让"我的听歌记录"变成不舍得删的资产；先发货后付款 |

组合使用示范——**先用锚定抬高参照点，再用限时制造损失厌恶，最后用试用触发禀赋效应**：这就是一个标准大促页的心理学配方。

### 应用的边界（重要）

这些机制是"放大器"而不是"发动机"：产品本身价值为正时，它们加速转化；产品价值为负时，它们制造一次性转化 + 长期反噬（差评、退款、卸载）。判断标准：**撤掉心理技巧后，用户是否仍然做出同样选择？** 若否，你在透支信任（呼应 [1.2 二阶思维](./1.2-通用心智模型.md)）。

### 示例：拼多多"砍一刀"的偏差全家桶

进度条直接从 90%+ 开始（锚定 + 目标临近效应）→ 每邀一人只涨 0.01%（已邀请 20 人的沉没成本让人停不下来）→ "即将过期，XX 元将消失"（损失厌恶：那笔钱在文案里早已"是你的"了——禀赋效应）。它是心理学工程化的极致样本，也是二阶效应（信任损耗）需要持续对冲的样本，两面都值得研究。

## 四、西奥迪尼影响力六原则

### 是什么

罗伯特·西奥迪尼《影响力》总结的六个说服原理，几乎覆盖所有运营转化文案与机制设计：

| 原则 | 机制 | 互联网应用示例 |
|------|------|--------------|
| **社会认同** | 不确定时跟随多数人 | 淘宝"已售 10万+"；大众点评"必吃榜"；直播间"3000 人正在抢" |
| **稀缺性** | 越难得到越想要 | "仅剩 2 件"；限量联名款；12306 候补的"余票紧张" |
| **承诺一致性** | 人倾向于与自己过去的公开承诺保持一致 | 先让用户选兴趣标签/立 Flag（Keep 的目标设定），后续以此推动行为；先答"你想省钱吗"再推月卡 |
| **互惠** | 先给予，对方产生回报压力 | 新人无门槛券；免费小样/试听课；瑞幸首杯免费 |
| **权威** | 专家/权威背书降低决策阻力 | "皮肤科医生推荐"；知乎高赞答主身份认证；官方检测报告 |
| **喜好** | 人更容易答应喜欢的人 | KOL/达人带货的底层逻辑；品牌人格化运营（老乡鸡微博） |

### 如何应用

1. **做转化文案自检**：一个商品详情页/活动页应至少覆盖 2-3 个原则（销量数字=社会认同，限时=稀缺，专家背书=权威），但堆满 6 个会显得像诈骗；
2. **顺序有讲究**：互惠放最前（先给再要），承诺一致性放流程早期（先小承诺后大请求，"登门槛"），稀缺放临门一脚；
3. **真实性红线**：伪造"仅剩 2 件"短期有效，被识破后该信号永久失效且连累品牌——稀缺必须真稀缺。

### 示例：直播带货 = 六原则的集装箱

头部主播的话术结构几乎是按教科书写的：粉丝亲密称呼（喜好）→ "我自己全家都在用"（承诺一致 + 权威）→ "全网最低价，品牌方亏钱"（互惠）→ "库存只有 5000 单"（稀缺）→ 实时滚动的"XX 已下单"（社会认同）→ "3、2、1 上链接"（稀缺的时间版）。理解了原理，你既能设计它，也能识别并抵御它。

## 五、心理账户与支付痛感

### 是什么

- **心理账户**（塞勒）：人不把钱看作同质的，而是分门别类放进不同"账户"——工资账户精打细算，红包/退税账户随手花掉；给孩子的教育账户不设上限，给自己的娱乐账户抠抠搜搜。
- **支付痛感**（Pain of Paying）：付钱的动作本身激活类似痛觉的神经反应；痛感强度与**支付的可见性、即时性、单次金额**正相关。现金 > 刷卡 > 手机支付 > 免密/自动扣款，痛感依次递减。

### 如何应用

| 策略 | 原理 | 示例 |
|------|------|------|
| 换账户叙事 | 把支出从"消费账户"挪到更宽松的账户 | "买会员 = 投资自己"（教育账户）；"一天不到一块钱"（零钱账户）；游戏皮肤定位为"社交形象"支出 |
| 预付/打包 | 支付一次痛一次，之后消费零痛感 | 视频年费会员；美团省钱月卡；健身年卡 |
| 代币化 | 用中间货币隔断"花钱"的感知 | 游戏先充值成"点券/钻石"再消费；直播平台的虚拟礼物 |
| 降低支付摩擦 | 减少支付动作的可见性 | 免密支付、先享后付、打车下车自动扣款（滴滴的体验拐点之一） |
| 拆痛为快 | 把大额支付拆小、把优惠即时显性化 | 分期免息；下单页"已省 ¥23"的绿色高亮 |

反向应用（帮用户省钱的产品）：**故意增加支付痛感**——记账 App 的大额消费提醒、支付宝的月度账单推送，本质是把被技术消除的痛感人为加回来。

### 示例：为什么打赏主播比给自己买书爽快

同样 100 元：买书走"学习消费账户"（要论证价值），打赏走"娱乐/社交账户"（要的就是即时爽感），且经过"金币/抖币"代币化，支付痛感被双重稀释。做变现设计时，先回答：**我的商品被用户放在哪个心理账户里？那个账户的额度和审批标准是什么？** 定价与话术都应围绕这个账户设计。

## 六、即时反馈与可变奖励

### 是什么

- **即时反馈**：行为之后奖励/反馈到来得越快，行为与结果的神经联结越强（操作性条件反射）。延迟一秒，强化效果都会显著衰减。
- **可变奖励**：斯金纳箱实验的核心发现——**不可预测的间歇性奖励，比固定奖励产生高得多且更难消退的行为频率**。多巴胺的峰值出现在"期待揭晓"的时刻，而非获得奖励的时刻。

Hook 模型中的三类可变奖励：
1. **部落型**（社交认同的不确定）：发朋友圈后反复回看有没有新赞；
2. **猎物型**（资源/信息的不确定）：刷信息流"下一条可能更精彩"、开盲盒、抽卡；
3. **自我型**（成就与掌控的不确定）：游戏升级、消灭收件箱、跑步破纪录。

### 如何应用

1. **给每个核心行为配即时反馈**：点赞的动画与音效、发布成功的强确认、下单后立刻可见的物流状态。反馈延迟是留存的隐形杀手——"操作了但没反应"等于告诉用户"你的行为无意义"；
2. **把固定奖励改造成可变奖励**：签到从"每天 10 金币"改为"随机 5-50 金币 + 第 7 天必得大奖"，成本相同，行为强化效果显著更高；
3. **三类奖励组合覆盖**：内容产品同时给猎物型（推荐流）、部落型（评论互动）、自我型（创作者数据），单一奖励类型的产品脆弱得多；
4. **伦理与合规红线**：可变奖励 + 付费 = 赌博机制的边缘。游戏抽卡在中国被要求公示概率；未成年人保护、防沉迷是硬约束。设计时自问：这个机制在帮用户完成他真正想做的事，还是仅在劫持他的多巴胺？

### 示例：抖音——史上最高效的猎物型可变奖励机

上滑手势 = 拉动老虎机摇杆：下一条视频的内容、质量完全不可预测，但算法保证"平均每几条就有一次命中"——这正是斯金纳箱中行为频率最高的**变比率强化**（Variable Ratio）程式。配合零延迟播放（即时反馈）与点赞/评论（部落型奖励），构成完整闭环。理解这一点，你就理解了"短视频刷起来停不下来"不是意志力问题，而是设计结果——以及为什么抖音要主动上线"时间管理"功能来对冲监管与舆论的二阶效应。

## 七、一张速查表：场景 → 该用哪个原理

| 工作场景 | 优先调用的原理 |
|---------|--------------|
| 注册/转化漏斗流失高 | Fogg：先砍步骤（A），再对时机（P），最后加动机（M） |
| 新功能没人用 | Fogg 的提示设计 + Hook 的触发设计 |
| 次日/7日留存差 | Hook 循环体检 + 即时反馈补齐 |
| 付费转化低 | 心理账户重定位 + 锚定定价 + 支付痛感优化 |
| 促销活动设计 | 稀缺 + 社会认同 + 损失厌恶（限时限量） |
| 会员续费/挽留 | 沉没成本 + 禀赋效应 + 损失厌恶话术 |
| 分享/裂变率低 | 部落型奖励 + 互惠（给被邀请者也发利益） |
