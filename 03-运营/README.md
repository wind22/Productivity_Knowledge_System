# 第3章 运营

> 📍 所属：第3章 运营 · 层级：导航 · 本章按「五层模型」组织运营知识：从底层原理到概念、方法、技巧、误区，帮你建立一套可迁移的运营知识体系。

## 本章定位

运营是互联网行业中最「杂」也最容易学散的岗位：拉新、留存、活动、内容、社群、商业化……如果只按业务模块罗列，学完仍然是一盘散沙。本章采用**五层模型**组织内容——先立原理（Why），再建概念（What），然后落方法（How），补充技巧（Hack），最后排雷误区（Trap）。建议按顺序通读一遍，之后按需查阅。
