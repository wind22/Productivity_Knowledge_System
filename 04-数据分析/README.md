# 第4章 数据分析

> 📍 所属：第4章 数据分析 · 层级：导航 · 一句话说明：本章按"原理→概念→方法→技巧→误区"五层模型组织，帮你建立从底层认知到实战落地的完整数据分析能力。

## 本章定位

数据分析是产品、运营、商业分析的共同底座：产品要用数据验证功能价值，运营要用数据评估活动增量，商业分析要用数据支撑经营决策。本章不追求面面俱到的教科书式覆盖，而是围绕互联网业务的高频场景（指标监控、异动排查、A/B 测试、用户分群、经营汇报），给出可直接指导工作的内容。

## 五层模型导航

| 层级 | 文档 | 回答的问题 | 核心内容 |
| --- | --- | --- | --- |
| 原理 | [4.1-第一性原理](./4.1-第一性原理.md) | 数据分析到底是什么、为什么有用？ | 用数据减少决策不确定性（信息→洞察→决策→行动）；相关≠因果；没有对比就没有分析；数据质量决定分析上限（GIGO） |
| 概念 | [4.2-核心概念](./4.2-核心概念.md) | 和数据打交道的基础词汇有哪些？ | 四组指标区分（绝对值/比率、存量/流量、先行/滞后、虚荣/北极星）；DAU、留存、LTV、CAC 等指标公式速查；统计学最小必备集（分位数、置信区间、p 值、辛普森悖论）；埋点与事件-属性模型；数仓分层（ODS/DWD/DWS/ADS） |
| 方法 | [4.3-方法论](./4.3-方法论.md) | 面对业务问题，按什么步骤分析？ | 指标体系搭建（北极星→OSM→指标字典）；指标拆解（乘法/加法/杜邦）；八大分析方法（对比、下钻、漏斗、留存、路径、分布、归因、Cohort）；异动分析五步 SOP；A/B 测试全流程与常见坑；用户分群（RFM/K-means/行为序列）；预测建模入门；经营分析从"报数"到"报观点" |
| 技巧 | [4.4-关键技巧](./4.4-关键技巧.md) | 同样的方法，怎么用出高水平？ | 先问业务问题再取数（需求澄清五问）；异常先怀疑数据本身（三查清单）；基线思维评估增量；可视化"一图一结论"；报告结论先行；SQL/Excel/pandas/BI/埋点平台工具箱 |
| 误区 | [4.5-常见误区](./4.5-常见误区.md) | 哪些坑最常见、如何自检？ | 虚荣指标代替健康指标；相关当因果、忽略混杂变量；A/B 样本不足与提前偷看；数据口径不统一——每个误区按「表现→为什么会踩→如何避免→自检问题」展开 |

## 推荐阅读路径

- **零基础入门**：4.1 → 4.2 → 4.4（先建立世界观和词汇，再学实战手感），需要时查阅 4.3 对应小节；
- **有经验进阶**：直接精读 4.3（方法论）+ 4.5（误区），用 4.5 的自检清单复盘自己过往的分析；
- **场景速查**：指标打架 → 4.5 第四节；指标突然异动 → 4.3 第四节 SOP；要做实验 → 4.3 第五节 + [AB测试记录表](../07-工具与模板库/表格/AB测试记录表.md)；要写汇报 → 4.4 第五节 + [分析报告模板](../07-工具与模板库/模板/分析报告模板.md)。
