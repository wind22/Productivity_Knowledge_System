# 第5章 商业分析

> 📍 所属：第5章 商业分析 · 层级：导航 · 本章帮你建立"看懂一门生意"的完整能力：从商业本质到财务拆解，从战略框架到实战技巧。

## 本章定位

产品经理、运营、数据分析师做到一定阶段，都会遇到同一个瓶颈：**只懂业务执行，不懂生意本身**。商业分析能力回答的是这类问题——

- 这个业务模式到底靠什么赚钱？账能不能算得过来？
- 竞争对手的财报透露了什么战略意图？
- 这个市场值不值得进？什么时候进？怎么进？
- 老板说"我们要做第二曲线"，到底在说什么？

## 五层模型导航

本知识库每一章都按"原理 → 概念 → 方法 → 技巧 → 误区"五层组织，由深到浅、由抽象到具体：

| 层级 | 文档 | 回答的问题 | 适合场景 |
|------|------|-----------|---------|
| 原理层 | [5.1 第一性原理](5.1-第一性原理.md) | 商业的本质是什么？利润从哪来？估值的底层逻辑是什么？ | 建立世界观，判断"这生意成立吗" |
| 概念层 | [5.2 核心概念](5.2-核心概念.md) | 商业模式画布、收入模式、单位经济、三张表、TAM/SAM/SOM、竞争格局 | 建立词汇表，看懂财报和 BP |
| 方法层 | [5.3 方法论](5.3-方法论.md) | 波特五力、安索夫矩阵、定价方法、竞对研究、投资视角分析 | 做一次完整的战略/竞对/市场分析 |
| 技巧层 | [5.4 关键技巧](5.4-关键技巧.md) | 快速算账、三步拆财报、竞对情报来源、BP 结构、敏感性分析 | 明天就要交付的实战任务 |
| 误区层 | [5.5 常见误区](5.5-常见误区.md) | 只看增长不看单位经济、TAM 拍脑袋、伪护城河、框架堆砌 | 自检与避坑 |
